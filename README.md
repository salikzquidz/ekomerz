# Ekomerz

An ecommerce web application where users can browse, search and purchase products that admin chose to display

## Goals

- [x] Have authentication system
- [x] Home Page
- [x] Slug Page
- [x] Guest Cart
- [x] User Cart
- [x] Checkout Page
- [ ] Order Details Page
- [ ] Admin Dashboard Page
- [ ] Admin upload Page
- [ ] Search bar

## Types of user

- Guest
- Buyer
- Admin

## DB Model

- Buyer
- Products
- Orders

## List of permissions

- view products
- add products
- delete products
- update products
- add item into cart
- update cart
- create order / checkout
- view orders
- update order's status
- delete order
